# Roadmap

## Initial development (v0.x.x)

- Experimentation, core features

## Stable release (v1.0.0)

- All main features: drawing, layers, PNG export

## Incremental features (v1.1.x, v1.2.x)

- Palette
- Optimized undo
- Improved UI
- GIF export
- Tileset & sprite sheet

## Major breaking change (v2.0.0)

- Canvas engine overhaul
- Animation (frame-based, onion skin)
- New JSON format
- Real-time collaboration mode
